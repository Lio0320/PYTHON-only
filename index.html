python Python 是一種廣泛使用的程式設計語言， 用於 Web 應用程式、軟體開發、資料科學與機器學習 (ML)。 開發人員使用 Python 在於效率高又容易學習， 並且可以在許多不同的平台上執行。Python 軟體可以免費下載， 與所有類型的系統完美整合，還可以加快開發速度。

print() 是 Python 的內建函數，用於將指定的內 容輸出到控制台（標準輸出設備）。 它可以輸出各種類 型的數據，包括字串、數字、列表、字典等。

 print("Hello, Python!")

input 是 HTML 表單元件裡最重要也最特別的元素， 主要的作用為「與使用者互動，讓使用者輸入資料」， 而輸入的方式又可分為直接打字輸入、點擊輸入或是下拉選單選擇...等，input 可以 根據類型 type 屬性的設定，轉換成不同的輸入型態 ( 例如輸入框、下拉選單、日期選單、按鈕...等 )。

 input("請輸入內容")
 input type="text"

if...else 是程式的條件判斷語法。當「如果」條件成立時，電腦會執行特定的程式碼；「否則」條件不成立時，則執行另一段預設程式碼。透過它，能讓電腦像人類一樣根據不同狀況做出自動應變與抉擇。

score = 75
if score >= 60:
    print("及格！恭喜通過。")
else:
   print("不及格，需要補考。")

list 回圈，最常見就是用 for 依序把清單裡的每個元素拿出來處理，語法像 for x in list_name:， 例如 fruits = ['apple', 'banana', 'orange']，for fruit in fruits: 就會把每個水果一個一個印出來。

fruits = ['apple', 'banana', 'orange']
for fruit in fruits:
   print(fruit)

range() 是 Python 裡用來產生一串整數的工具，最常搭配 for 迴圈使用來控制執行次數；基本寫法有 range(stop)、range(start, stop)、 range(start, stop, step)，而且 stop 不會被包含。例如 for i in r ange(5): 會跑 0 到 4，for i in range(2, 8): 會跑 2 到 7，for i in range (10, 0, -2): 會跑 10、8、6、4、2。

for i in range(3):
   print(i)

break 是 Python 裡用來「立刻跳出整個迴圈」的關鍵詞，只要是 for 或 while 迴圈，當執行到 break，就會直接結束整個迴圈，不再繼續執行剩下的次數或條件。

nums = [1, 3, 5, 7] target = 4
for n in nums:
   if n == target:
      print("找到目標")
      break
   else:
      print("沒有找到目標")
